##  Projetos

###  [Chatbot de Agendamento - Barbearia e Clinicas](https://github.com/Kaua-tech1/automation-de-Agendamento-n8n-whatsapp)
Sistema completo de atendimento via WhatsApp com dois agentes de IA em cadeia.

- Agente principal (o4-mini) gerencia a conversa e contexto
- Sub-agente especializado (GPT-4.1) verifica, registra e cancela agendamentos no Google Calendar
- Transcrição de áudio via Groq Whisper
- Memória de sessão com Redis + persistência no Supabase
- Debounce, rate limiting e bloqueio anti-loop

**Stack:** n8n · OpenAI · Redis · Supabase · Google Calendar · Evolution API

---

###  [Inventário Empresarial por WhatsApp](https://github.com/Kaua-tech1/automation-de-inventario-n8n-whatsapp)
Sistema de gestão de inventário operado inteiramente via WhatsApp, sem necessidade de interface.

- Funcionários registram entradas e saídas de produtos em linguagem natural
- Agente de IA (Gemini) interpreta as mensagens e gera saída estruturada
- Cache de produtos no Redis para respostas rápidas
- Fluxo de confirmação em etapas antes de gravar no banco
- Alertas automáticos ao gerente quando itens atingem nível crítico

>  Em desenvolvimento — migração para nós nativos em andamento

**Stack:** n8n · Gemini · Redis · Supabase · Evolution API
